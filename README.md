lyx2ebook
=========
0.1.0 alpha  


What is lyx2ebook
-----------------

As its name implies, it is a simple LyX to eBook converter.

It is written in Python 2.6, and it requires the lepl module.



Features
--------

Not much yet.



Usage
-----

Convert LyX document to ePub file:

    lyx2epub simple.lyx
